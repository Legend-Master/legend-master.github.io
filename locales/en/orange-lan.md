[Orange LAN](https://orange-lan.atadg.com/) is a [SaaS](https://en.wikipedia.org/wiki/Software_as_a_service) I built for the Chinese market that allows you to easily create virtual Local Area Networks to improve latency of multi-player games that supports LAN gaming or uses [Steam Networking](https://partner.steamgames.com/doc/features/multiplayer/networking)

What differentiates Orange LAN from the other VPNs is that we hide all the technical details well enough that the user will never need to understand all the networking concepts to use it. The entire process is done by you simply clicking on a button to create a room and let the other person join it through the room ID, then you can play your games like normal and **it just works**.

The app features a super small bundle size (~1MB), while having a modern look with all the UX details included such as page transitions, auto focus on input box when you type, etc.

Take a look at our website https://orange-lan.atadg.com/ if you're interested
